## 文本样式

也可以组合：***粗体加斜体***，**粗体中的 `代码`**。

## 列表

### 无序列表

- 第一项
- 第二项
  - 嵌套子项 A
  - 嵌套子项 B
    - 更深层级
- 第三项

### 有序列表

1. 第一步：安装依赖
2. 第二步：配置环境
3. 第三步：启动服务
   1. 检查端口占用
   2. 运行启动命令

### 任务列表

- [x] 已完成的任务
- [x] 另一项已完成
- [ ] 待办事项
- [ ] 另一项待办

## HTML 内嵌

Markdown 支持直接嵌入 HTML：

<details>
<summary>点击展开详情</summary>

这里是被折叠的内容，可以包含 Markdown 格式。

- 列表项
- **粗体**

</details>
